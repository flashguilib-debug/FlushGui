FlashGui · Costruisci interfacce desktop con il sorriso

Una libreria Python che avvolge wxPython in un abbraccio moderno: componenti già pronti, layout proporzionale, temi personalizzabili e una sintassi che non ti farà impazzire.

✨ Perché FlashGui?

Hai presente quando vuoi creare una piccola app desktop in Python e ti scontri con la complessità di Tkinter o la verbosità di wxPython? FlashGui nasce per darti calcolatrice, blocco note, paint, orologio e calendario già funzionanti in 3 righe di codice. Ma se vuoi costruire la tua interfaccia da zero, il componente WINDOW ti permette di posizionare qualsiasi widget usando coordinate proporzionali (0.0 → 1.0) e i font si ridimensionano da soli. Tutto con un look pulito, moderno e completamente personalizzabile.

💡 Idea: FlashGui è perfetto per prototipi rapidi, tool interni, o per chi si avvicina alla programmazione GUI e vuole risultati immediati senza studiare mille manuali.

📦 Installazione in un soffio

pip install FlashGui

wxPython e charset_normalizer verranno installati automaticamente. Nessun'altra dipendenza.

✅ Funziona su: Windows 10/11, macOS (Intel/Apple Silicon), Linux con wxPython supportato. Richiede Python 3.7 o superiore.

📚 Dipendenze

FlashGui è costruito sopra wxPython Phoenix (wxWidgets) e utilizza charset_normalizer per il rilevamento automatico delle codifiche nel blocco note. Entrambe le librerie vengono installate automaticamente con pip.

🧮 CALCULATOR – La calcolatrice che puoi vestire a modo tuo

Una calcolatrice completa con display, 20 pulsanti e gestione degli errori. Puoi cambiarne ogni singolo aspetto e agganciare le tue funzioni a ogni tasto.

  • Clear() → svuota display e memoria.

Metodi per manipolare i widget

Poiché ogni metodo Add... restituisce l'oggetto, puoi conservarlo in una variabile e usare i metodi nativi di wxPython. In alternativa, FlashGui offre alcuni helper:

  • EnableWidget(widget, enable=True) → abilita/disabilita un widget
  • ShowWidget(widget, show=True) → mostra/nasconde un widget (e ricalcola il layout)
  • CanvasSetPen(canvas, color=None, width=None) → imposta colore/spessore per il canvas
  • CanvasClear(canvas, color=None) → pulisce il canvas
💡 Esempio: btn = win.AddButton("Cliccami", relx=0.4, rely=0.4, relwidth=0.2, relheight=0.1)
Poi puoi fare btn.SetLabel("Nuovo testo") o win.EnableWidget(btn, False).

Menu e barra di stato

  • AddMenuBar() (automatico se si aggiunge un menu)
  • AddMenu(title) → restituisce un menu wx.Menu
  • AddMenuItem(menu, label, command, shortcut) → aggiunge una voce
  • AddStatusBar(testo_iniziale)
  • SetStatusText(testo, field=0)

Utility integrate

  • AddTimer(ms, command) → esegue la funzione ogni ms millisecondi
  • Message(testo, titolo="Notice") → mostra un messaggio
  • AskSave(wildcard, default_dir) → restituisce percorso o None
  • AskOpen(wildcard, default_dir) → idem
  • exit() → chiude l’applicazione

🔁 La triade magica: run(), Edit(), Logic()

Ogni componente FlashGui espone questi tre metodi fondamentali:

  • .run() – Avvia il loop principale dell'applicazione (bloccante).
  • .Edit(**kwargs) – Modifica l'aspetto: testi, colori, messaggi. Ogni componente ha i suoi parametri specifici (elencati sopra).
  • .Logic(**kwargs) – Collega le tue funzioni agli eventi (click pulsanti, selezione voci di menu, ecc.).
🧠 Ordine consigliato: Crea l'oggetto → .Edit(...) → .Logic(...) → .run().

📚 Esempi pratici – Impara facendo

1. Calcolatrice con memoria (CALCULATOR)

calc_memory.py
from FlashGui import CALCULATOR

memoria = 0

def memorizza():
    global memoria
    try:
        memoria = float(calc.GetText())
    except:
        pass

def richiama():
    calc.SetText(str(memoria))

calc = CALCULATOR(title="Calc con Memoria", size=(2.5,2))
calc.Edit(button_0_text="M+", button_1_text="MR")
calc.Logic(button_0_command=memorizza, button_1_command=richiama)
calc.run()

2. Blocco note minimale (NOTEPAD)

simple_notepad.py
from FlashGui import NOTEPAD

def quando_salvo():
    print("File salvato!")

note = NOTEPAD(title="Il mio diario", size=(1.5, 1.2))
note.Edit(
    text_ctrl_bg=(255, 255, 200),   # sfondo giallo pastello
    text_ctrl_fg=(50, 50, 150),
    save_text="Salva 💾"
)
note.Logic(menu_save_command=quando_salvo)  # override del salvataggio
note.run()

3. Paint con strumenti personalizzati (PAINT)

my_paint.py
from FlashGui import PAINT

p = PAINT(title="Disegna qualcosa", size=(1.8, 1.5))

4. Svezia – Timer con allarme (CLOCK)

timer_sveglia.py
from FlashGui import CLOCK

def fine_timer():
    win.Message("Il tempo è scaduto!", "Sveglia")

orologio = CLOCK(title="Timer")
orologio.Logic(timer_start_command=fine_timer, timer_base_m=1)  # 1 minuto
orologio.run()

5. Calendario con eventi (CALENDAR)

eventi_calendar.py
from FlashGui import CALENDAR

def giorno_selezionato(data):
    win.set_info_text(f"Hai scelto il {data}")

win = CALENDAR(title="Agenda")
win.Logic(day_command=giorno_selezionato, week_start_monday=True)
win.run()

6. Costruttore di interfacce (WINDOW)

widget_demo.py
from FlashGui import WINDOW

def saluta():
    nome = campo_input.GetValue()
    win.Message(f"Ciao {nome}!")

win = WINDOW(title="Demo interattiva", size=(1.5, 1.2))

# Aggiungiamo un'etichetta
win.AddLabel("Come ti chiami?", relx=0.2, rely=0.2, relwidth=0.6, relheight=0.1)

# Campo di testo
campo_input = win.AddInput(relx=0.2, rely=0.35, relwidth=0.6, relheight=0.1)

# Pulsante
win.AddButton("Saluta", command=saluta, relx=0.35, rely=0.6, relwidth=0.3, relheight=0.1)

# Barra di stato
win.AddStatusBar("Pronto")

win.run()
🔍 Nota: In WINDOW, i widget vengono manipolati tramite le variabili che li referenziano (es. campo_input). Non esistono metodi GetWidget(name); usa direttamente l'oggetto restituito da Add....
